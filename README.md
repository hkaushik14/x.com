# X.com Clone

This project is a frontend clone of X.com (formerly Twitter) built for learning and practice purposes.

## Tech Stack
HTML, CSS, Tailwind CSS, JavaScript, Vite

## How to Run the Project
1. Clone the repository using  
git clone https://github.com/hkaushik14/x.com.git  

2. Move into the project folder  
cd x.com  

3. Install all dependencies  
npm install  

4. Start the development server  
npm run dev  

Open your browser and visit:  
http://localhost:5173  

## Note
The node_modules folder is ignored using .gitignore and will be generated automatically after running npm install.
